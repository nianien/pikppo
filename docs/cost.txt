一、逐项确认（结论：你的判断是对的）

✅ 1. Google Cloud Speech-to-Text API（必需）

用途
	•	中文 ASR
	•	说话人分离（Diarization） ← 这是关键价值

为什么这是 v1 的正确选择
	•	在线
	•	工程成熟
	•	diarization 准确率高于“Whisper + 本地拼装”
	•	直接产出 speaker_tag，省掉一个大模块

环境变量

GOOGLE_APPLICATION_CREDENTIALS=/path/to/credentials.json

成本校正
	•	标准模型：约 $0.006 / 15 秒
	•	即 $0.024 / 分钟
	•	一集 2 分钟 ≈ $0.048
	•	前 60 分钟/月免费 ✅

✔️ 确认：必需，且性价比极高

⸻

✅ 2. OpenAI API（必需）

用途
	•	两阶段翻译：
	•	Episode Context（人物/称谓/术语统一）
	•	Segment Translation（逐句、长度受控）

为什么不用传统翻译 API
	•	短剧是：
	•	高情绪
	•	强上下文
	•	口语化
	•	传统 MT（Google Translate）在这个场景不够用

环境变量

OPENAI_API_KEY=sk-xxxx

成本校正（以 gpt-4o-mini 为例）
	•	输入：$0.15 / 1M tokens
	•	输出：$0.60 / 1M tokens

现实换算
	•	一集短剧（含上下文）≈ 3k–6k tokens
	•	成本 ≈ $0.003–0.01 / 集

✔️ 确认：必需，而且非常便宜

⸻

✅ 3. Azure Speech Service（TTS）（必需）

用途
	•	英文 TTS
	•	8 条固定美式英语声线池
	•	稳定、可控、批量友好

环境变量

AZURE_SPEECH_KEY=xxxx
AZURE_SPEECH_REGION=eastus

成本校正
	•	官方价：≈ $16 / 1M 字符
	•	前 50 万字符/月免费

现实换算
	•	1 分钟英文对白 ≈ 900–1200 字符
	•	一集 2 分钟 ≈ 2k 字符
	•	成本 ≈ $0.03 / 集
	•	免费额度可跑 ~250 集/月

✔️ 确认：必需，且比 ElevenLabs 更适合 v1
