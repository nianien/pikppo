# 念音助理 · App 图标说明

## 设计

主色 `#3FB984`(清新春绿)圆角方底,三片白色饱满柳叶以 120° 旋转对称围合。

三片叶之间留出的负形缝隙,从中心向外辐射,自然构成一个字母 **Y**。

## 寓意

- **三叶草** —— 生长、清新、幸运;呼应"陪伴在身边"的个人助理定位。
- **负形 Y** —— Y 是「念音」拼音 nián**y**īn 的中段,也象征分支、连接与倾听;藏在叶与叶的留白里,不喧宾夺主,细看方见,符合「念音」克制、含蓄的气质。
- **三片叶** 亦可引申为助理的三种能力:**知**(懂你)、**忆**(记得)、**陪**(陪伴)。

## 文件清单

- `nianyin-icon.svg` —— 主图标矢量源文件(含圆角绿底),1024×1024
- `nianyin-mark-transparent.svg` —— 透明背景标志(仅绿色图形),用于组合标、网页、深色场景
- `icons-ios/` —— iOS 全套尺寸 PNG(20–1024)
- `icons-android/` —— Android mipmap 各密度 + Play Store 512 PNG
- `mark-*.png` —— 透明标志常用尺寸

## 用色规范

| 用途 | 色值 |
|---|---|
| 主底色 | `#3FB984` |
| 图形(底色上) | `#FFFFFF` |
| 浅底变体 | 底 `#E4F4EC` / 图形 `#2E9E6E` |
| 白底变体 | 底 `#FFFFFF` / 图形 `#3FB984`(描边 1.2px) |

## 使用注意

- 圆角半径已按 iOS/Android 规范设置(1024 下 rx=230),系统会自动套用各自圆角遮罩,导出 PNG 保留原始方形即可。
- 最小使用尺寸建议不低于 24px;低于此尺寸负形 Y 会糊,但三叶草轮廓仍清晰。
- 切勿拉伸变形、改变叶片角度或旋转对称关系。
- 单色场景(印刷、水印)直接用透明 mark 改 fill 即可。
